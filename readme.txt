This plugin adds the MobileCoin Payment Method on WooCommerce Checkouts.

== Description ==

This plugin is designed to add the MobileCoin Payment Method on WooCommerce Checkouts.
If selected, the payment method will redirect the clients on the MobileCoin Platform where they'll be able to pay with MOB.



== How to get your MobileCoin API Keys ==

Send us an email at payments@mobilecoin.com and we'll take care of it.
